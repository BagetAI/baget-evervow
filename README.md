# EverVow | Sustainable Luxury Secondhand Bridal

A curated marketplace for designer secondhand wedding dresses, vertically integrated with master tailors for a guaranteed fit.

## Tech Stack
- **Frontend**: React (via CDN)
- **Styling**: Tailwind CSS
- **Database**: Baget Agent Database (EverVow Waitlist)
- **Deployment**: Vercel

## Key Features
- **Dual-Sided Experience**: Dedicated pathways for Buyers (Find Your Fit) and Sellers (Sell Your Magic).
- **Waitlist Engine**: Integrated with internal databases for real-time lead capture.
- **Retro Analog Aesthetic**: Heritage-inspired design following established brand guidelines.
